01 O combustível

Cada token que a IA queima é um tijolo. O app lê teu uso real do Claude Code ao vivo; este preview no navegador só simula.

02 A cidade não para

A cada leva de tokens nasce um prédio — e ela se estende pela costa sem fim: os antigos deslizam pra esquerda enquanto novos surgem.

03 O dia passa

Entardece, anoitece e amanhece em ciclo. À noite as janelas acendem uma a uma e a cidade se reflete na água.

04 No cantinho

Feito pra viver como janela flutuante no canto da tela — a versão de app (Electron) roda sempre por cima enquanto você trabalha.

Protótipo. Tokens simulados só pra você sentir o clima. Rode npm start na pasta pra ver flutuando de verdade.
